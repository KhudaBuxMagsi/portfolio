- Modern responsive UI
- AI Engineer profile
- Skills section

## Tech Stack

- React
- TypeScript
- Vite
- CSS3
- Gemini API (if enabled)

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Environment Variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_api_key_here
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
